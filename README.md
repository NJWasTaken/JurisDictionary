# JurisDictionary
GenAI project

# Usage

- First clone this repo.
```git clone https://github.com/nishantHolla/JurisDictionary.git```

- Create python environment
```python -m venv env```

- Install python dependencies
```
pip install -r /path/to/requirements.txt
```

- Run the app
```
streamlit run ./app.py
```
