# JurisDictionary
GenAI project

# Usage

- First clone this repo.<br>
```git clone https://github.com/nishantHolla/JurisDictionary.git```

- Create python environment<br>
```python -m venv env```

- Install python dependencies<br>
```
pip install -r /path/to/requirements.txt
```

- Run the app<br>
```
streamlit run ./app.py
```
